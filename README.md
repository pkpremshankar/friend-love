# 💖 Friend Love – Interactive Apology Website

An emotional and interactive website created to express feelings and say sorry in a unique and beautiful way. This project combines design, animation, and heartfelt messaging to create a memorable experience.

---

## ✨ Features

* 💌 Emotional apology message
* ❤️ Beautiful UI with soft colors and clean layout
* 🎯 Interactive buttons (Yes / No response)
* 🌐 Smooth navigation between pages
* 📱 Fully responsive design

---

## 🚀 Live Demo

👉 https://friend-love.netlify.app/

---

## 📁 Project Structure

* `index.html` – Main webpage
* `love besti.code-workspace` – VS Code workspace file

---

## 💡 How It Works

1. User opens the website
2. A heartfelt apology message is displayed
3. User clicks “आगे बढ़ें” (Next)
4. A question screen appears
5. User can respond with “हाँ” or “नहीं”
